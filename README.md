## 1. Fibonacci Series

### Problem

Write a program to print the Fibonacci series up to N terms.

### Approach

The Fibonacci sequence is generated by adding the previous two numbers.

Steps:

1. Start with two numbers: 0 and 1
2. Print the first number
3. Add the previous two numbers to generate the next number
4. Repeat the process for N terms

### Code

```javascript
function fibonacci(n) {
    let a = 0, b = 1;

    for (let i = 1; i <= n; i++) {
        console.log(a);
        let c = a + b;
        a = b;
        b = c;
    }
}

fibonacci(10);
```

## 2. Sum of Digits of a Number

### Problem

Write a program to find the sum of digits of a number.

### Approach

1. Take the number as input
2. Extract the last digit using `% 10`
3. Add the digit to a sum variable
4. Remove the last digit using `Math.floor(n / 10)`
5. Repeat until the number becomes 0

### Code

```javascript
function sumOfDigits(n) {
    let sum = 0;

    while (n > 0) {
        sum += n % 10;
        n = Math.floor(n / 10);
    }

    console.log("Sum of digits:", sum);
}

sumOfDigits(1234);
```

## 3. Palindrome Number

### Problem

Check whether a number is palindrome or not.

### Approach

1. Reverse the digits of the number
2. Compare the reversed number with the original number
3. If both numbers are equal, the number is a palindrome

### Code

```javascript
function palindrome(num) {
    let original = num;
    let reversed = 0;

    while (num > 0) {
        let digit = num % 10;
        reversed = reversed * 10 + digit;
        num = Math.floor(num / 10);
    }

    if (original === reversed)
        console.log("Palindrome");
    else
        console.log("Not Palindrome");
}

palindrome(121);
```

## 4. Triangle Pattern

### Problem

Print the following triangle pattern:

*

**

---

---

### Approach

1. Use two loops
2. The outer loop controls the number of rows
3. The inner loop prints the stars for each row

### Code

```javascript
function triangle(n) {
    for (let i = 1; i <= n; i++) {
        let row = "";

        for (let j = 1; j <= i; j++) {
            row += "*";
        }

        console.log(row);
    }
}

triangle(4);
```

## 5. Factorial

### Problem

Write a program to calculate the factorial of a number.

### Approach

Factorial of a number is calculated as:

n! = n × (n-1) × (n-2) × ... × 1

Steps:

1. Take the number as input
2. Multiply numbers from 1 to n
3. Store the result in a variable

### Code

```javascript
function factorial(n) {
    let fact = 1;

    for (let i = 1; i <= n; i++) {
        fact *= i;
    }

    console.log("Factorial:", fact);
}

factorial(5);
```
